Abstract

1. Introduction
1.はじめに

As of this writing, the BGP Path Attributes registry maintained by IANA contains entries for DPA, ADVERTISER, and RCID_PATH / CLUSTER_ID. The first of these is associated with [BGPDPA], an Internet-Draft that was abandoned in 1996. The latter are associated with [RFC1863], an RFC that was reclassified as Historic by [RFC4223] in 2005. Neither specification is currently in use, nor have they ever been.

この記事の執筆時点で、BGPのパスは、IANAによって維持レジストリはDPA、広告主、およびRCID_PATH / CLUSTER_IDのエントリが含まれ属性。これらの最初は1996年に後者を放棄したBGPDPA]、インターネットドラフトに関連付けられているいずれの仕様が現在使用中である2005年に[RFC1863]、[RFC4223]で歴史として再分類されたRFCに関連付けられています、また彼らがこれまでされています。

Note that the deprecated CLUSTER_ID is distinct from CLUSTER_LIST [RFC4456]. CLUSTER_LIST is not being deprecated.

非推奨CLUSTER_IDがCLUSTER_LIST [RFC4456]とは異なることに注意してください。 CLUSTER_LISTは廃止されていません。

2. IANA Considerations
2. IANAの考慮事項

Per this document, IANA has marked the BGP Path Attributes registry entries for DPA (Value 11), ADVERTISER (Value 12), and RCID_PATH / CLUSTER_ID (Value 13) as "deprecated".

このドキュメントごとに、IANAは、BGPパスは "非推奨" とDPA（値11）は、広告主（値12）、及びRCID_PATH / CLUSTER_ID（バリュー13）のレジストリエントリの属性マークしました。

3. Security Considerations
3.セキュリティの考慮事項

There are no meaningful security consequences arising from this registry update.

このレジストリの更新から生じる意味のあるセキュリティ上の影響はありません。
